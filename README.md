_Calorie Counter is an application that uses the [ResNet50](https://developer.apple.com/machine-learning/models/) core  model to detect food images that you can either click or upload from your gallary and helps you detect the number of calories the food item might have and helps you maintain your desired calorie count for the day_ 

## Features

 |         | Existing Features  |
----------|-----------------
:corn: | List of common food items
:white_check_mark: | Clicking a picture of the food item
:mag: | Search feature to find from the list of food items
:open_file_folder: | Calories remaining for the day

 |         | Future Features  |ML
----------|-----------------
:walking: | Calories exerted using Step Count
:iphone: | Input Calories using Siri Shortcut
:heavy_exclamation_mark: | Reminders to eat healthy through Notifications
